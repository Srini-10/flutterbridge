## Compatibility: Low Compatibility

The application depends on capabilities that do not exist in a browser at all — web-incompatible imports (dart:io). These are not gaps in the compiler; they are gaps in the target platform. Converting this application means a human first decides what those capabilities become on the web. The UI layer itself may still convert well — see the supported/partial lists.

## Warnings

### W06 — CustomPainter / CustomPaint

**Why.** Imperative canvas drawing. Spec §7 names this as the automation boundary.

**Impact.** The painted widget cannot be generated; it needs a hand-written React component.

**Likely compiler behaviour.** Emit an Opaque node keyed by a stable anchor and route it to the override system, so the human writes it once and regeneration preserves it.

**Owner.** generator (override system)

**Where.** `lib/ui/common/dashed_line.dart:12`, `lib/ui/common/dashed_line.dart:17`, `lib/ui/screens/artifact/artifact_search/time_range_selector/expanding_time_range_selector.dart:258`, `lib/ui/screens/artifact/artifact_search/time_range_selector/time_range_painter.dart:8`

### W07 — Explicit animation (AnimationController / TickerProvider)

**Why.** Explicit animation choreography is outside the MVP; only implicit animations are planned to map cleanly.

**Impact.** Transitions driven by a controller will not be reproduced.

**Likely compiler behaviour.** The animation runtime handles the implicit cases; a controller timeline flags for an override.

**Owner.** runtime (animation engine)

**Where.** `lib/ui/common/modals/fullscreen_url_img_viewer.dart:138`, `lib/ui/screens/editorial/widgets/_circular_title_bar.dart:75`, `lib/ui/screens/editorial/widgets/_circular_title_bar.dart:79`, `lib/ui/screens/editorial/widgets/_section_divider.dart:18`, `lib/ui/screens/home/_vertical_swipe_controller.dart:5`, `lib/ui/screens/home/wonders_home_screen.dart:30`, `lib/ui/screens/wonder_details/wonders_details_screen.dart:19`, `lib/ui/wonder_illustrations/common/animated_clouds.dart:27`, `lib/ui/wonder_illustrations/common/animated_clouds.dart:30`, `lib/ui/wonder_illustrations/common/wonder_illustration_builder.dart:28`, `lib/ui/wonder_illustrations/common/wonder_illustration_builder.dart:29`

### W08 — Web-incompatible import: dart:io

**Why.** dart:io is not available on the web at all.

**Impact.** Any code path reaching this import cannot run in a browser, converted or not.

**Likely compiler behaviour.** Refuse to convert the enclosing unit; require an override or a web-specific implementation.

**Owner.** analyzer (detect) + bridge_lints (prevent, for new projects)

**Where.** `lib/_tools/artifact_download_helper.dart:2`, `lib/_tools/artifact_search_helper.dart:6`, `lib/_tools/unsplash_download_service.dart:2`
